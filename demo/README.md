# Frontend Style Demos

本目录用于对比 Aindexer 正式产品前端的候选设计方向。

## 当前基线

- `editorial-lab/` 已定为 **V3.5 前端设计基线**。
- V3.5 不再追求复刻 V2，而是以学术编辑台、暖色纸张感、阅读与修订优先为主方向。
- `research-os/` 保留为备选参考，不作为当前迁移目标。

## 方案

- `research-os/FRONTEND_DESIGN.md`：浅色科研工作台，偏仪器面板和高密度操作。
- `research-os/index.html`：Research OS 静态页面示例。
- `editorial-lab/FRONTEND_DESIGN.md`：V3.5 前端设计基线，暖色学术编辑台，偏阅读、修订和翻译校对。
- `editorial-lab/index.html`：V3.5 静态页面示例。

## 查看方式

直接在浏览器打开对应目录下的 `index.html` 即可。两个示例均为纯静态 HTML，不依赖构建工具或后端服务。

## 判断建议

- 如果优先目标是高效操作、批量索引、状态管理和长期工具感，优先看 Research OS。
- 如果优先目标是阅读氛围、人工修订、翻译校对和学术编辑感，优先看 Editorial Lab。
